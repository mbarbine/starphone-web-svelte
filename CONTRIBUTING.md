# Contributing

Thanks for contributing.

## Development setup

```bash
# install dependencies
pnpm install

# run checks
pnpm lint
pnpm test
pnpm build
```

If the project uses npm, use `npm` instead of `pnpm` in those commands.

## Pull request process

1. Create a small, focused change.
2. Include test updates and verification evidence.
3. Submit a PR with a short summary and related issue number.
